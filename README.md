# Game Launcher & Library Manager
A desktop application built with WPF (Windows Presentation Foundation) and .NET designed to organize and launch PC games. This project demonstrates the implementation of a modern UI, local file management, and persistent data storage.

🚀 Features
Game Library Management: Add, edit, and remove games from your local database.

Persistent Storage: Uses SQLite (or Local Storage) to keep your library saved between sessions.

Time Tracking: Real-time monitoring of "Time Played" for each launched title.

Visual Progress: Interactive progress bars and status indicators for game updates or installation states.

Custom UI: A dark-themed, modern interface inspired by popular gaming platforms like Steam and GOG Galaxy.

🛠️ Built With
C# / .NET - Core application logic.

WPF (XAML) - Rich desktop user interface.

Entity Framework Core - Database management and ORM.

SQLite - Lightweight, serverless local database.

📈 Key Learnings
This project was developed to master several software engineering concepts:

CRUD Operations: Implementing Create, Read, Update, and Delete functionality within a desktop environment.

Event Handling: Managing process starts and exits to track application usage time.

Data Binding: Utilizing the MVVM (Model-View-ViewModel) pattern to sync the UI with underlying data models.
